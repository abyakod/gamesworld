# AB'S Game Zone: Logic & Visual IQ

A streamlined, premium game hub focused on developing logic, spatial reasoning, and visual pattern recognition.

## 🎮 Games Included

### 🔍 Catch the Difference
Strengthens **Visual IQ** and **Pattern Recognition**.
- **Increased Difficulty**: High item density (25+ objects) and subtle differences.
- **Tough Challenges**: Includes emoji swaps, 180-degree rotations, and size shifts.
- **Progressive Scaling**: Differences needed increase as you level up.

### 🧪 Science Lab (Water Sort)
Develops **Logic and Planning** (Spatial Reasoning).
- Rebranded version of the classic liquid sorting puzzle.
- Enhances multi-step problem-solving skills with hidden segments and adaptive difficulty.

## ✨ Features
- **Clean UI**: A focused, "plain" interface designed for maximum engagement without distractions.
- **Global Rewards**: Earn stars across games to track your progress.
- **Premium Aesthetics**: Smooth animations and vibrant colors.

## 🚀 Getting Started
To test the games locally:
1. Open `index.html` in any modern web browser.
2. Or run a local server:
   ```bash
   python3 -m http.server 8000
   ```
3. Navigate to: `http://localhost:8000`

## 🌐 Deployment
This project is ready to be hosted on **GitHub Pages**:
1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Select the `main` branch and click **Save**.
