# Kids Learning Hub: Mind Improvement Games

A premium, interactive game hub designed for kids aged 5-7 to practice math, spelling, and logic through engaging gameplay.

## 🎮 Games Included

### 🔢 Math Adventure
Builds **Quantitative Reasoning** through addition and subtraction practice.
- Features visual counting aids (stars) to bridge the gap between numbers and quantities.
- Adaptive difficulty.

### 🔤 ABC Explorer
Strengthens **Linguistic Pattern Recognition** and spelling.
- Uses emoji-based visual hints.
- Scrambled letter tiles to encourage sequencing and recall.

### 🧪 Science Lab
Develops **Logic and Planning** (Spatial Reasoning).
- Rebranded version of the classic liquid sorting puzzle.
- Enhances multi-step problem-solving skills.

## ✨ Features
- **Vibrant UI**: Premium aesthetics with smooth animations.
- **Global Rewards**: Earn stars across all games to track progress.
- **Cognitive Focus**: Designed to improve memory, focus, and logic.

## How to Host on GitHub Pages (Free Internet Hosting)

To share this game with anyone on the internet, host it for free using GitHub Pages:

1. **Initialize Git**:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Water Sort Game"
   ```
2. **Create a GitHub Repository**:
   - Go to [GitHub.com](https://github.com) and create a new public repository (e.g., `water-sort-game`).
   - Do NOT initialize it with a README (leave it completely empty).
3. **Push the Code**:
   ```bash
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/water-sort-game.git
   git push -u origin main
   ```
4. **Enable GitHub Pages**:
   - On GitHub, go to your repository's **Settings** tab.
   - Click **Pages** on the left sidebar.
   - Under "Build and deployment", set the **Source** to "Deploy from a branch".
   - Select the `main` branch and `/ (root)` folder, then click **Save**.
   
Within a few minutes, your game will be live at: `https://YOUR_USERNAME.github.io/water-sort-game/`
